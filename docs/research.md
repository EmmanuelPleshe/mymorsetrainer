## Key Insights

- **k3ng_cw_keyer**: 495 stars, audio timing, key input decoding, modular Arduino architecture
- **ecwt**: Android audio handling, Kotlin, available on F-Droid
- **morse-code-trainer**: Python audio with pygame, keyboard input handling
- **cw-trainer**: Koch method at 20-30 WPM, Arduino-based

## License Notes

- GPL-3.0 (k3ng): Must share source modifications
- MIT (ecwt, mjago/CW, morse-code-trainer): Very permissive, can learn from freely
- No license (cw-trainer): Contact author before using code

## Inspiration

- Don Dodge W6SNX brought Arduino CW trainer + straight key to Lake Washington Ham Club Rain or Shine gathering
- Goal: Interactive keying loop (hear → key back → verify → feedback)
